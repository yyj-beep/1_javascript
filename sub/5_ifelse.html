Conditional Statements

 

조건문 
- if : 지정된 조건이 참인 경우 실행할 코드블록을 지정하는데 사용합니다
syntax
if(condition){}
- else : 동일한 존건이 거짓인 경우 실행할 코드블록을 지정하는데 사용
- else if : 첫번째 조건이 거짓인 경우 실행할 코드블록을 지정하는데 사용
- switch : 실행할 여러 대체 코드블록을 지정하는데 사용

switch : 조건이 여러개일 경우 깔끔한 코딩을 위해 사용
switch(expression) {
    case x:
    코드블록
    break;
}
