Operators

- 할당연산자  = 변수에 값을 할당합니다 => let x = 10;
- 덧셈 연산자 + 는 값을 더합니다 => let x = 5; let y=2; let z = x + y
- 곱셈 연산자 * 는 값을 곱합니다 => let q = 5; let w=5; let multi = w * q
- 비교 연산자 > 값을 비교합니다 => 변수를 선언후 값을 비교합니다
자바스크립트 연산자의 유형
산술연산자[+,-,*,**,/,%[나눈 나머지는],++,--],할당연산자,비교연산자,문자열연산자,논리 연산자,비트 연산자,삼항 연산자

비교연산자 true, false

- == 같음인데 엄격하게 같음이 아님
- === 타입까지 고려하여 엄격하게 같음
- > ,<

Logical operators : 논리연산자

- && => 그리고 조건이 두개가 일치해야함 true
- || => or 둘중에 하나라도 맞으면 true
- ! => 같지않음 부정이 진실이고 진실이 거짓이다
