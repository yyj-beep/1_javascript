- onchange : HTML요소가 변경될때
- onclick : 사용자가 html 엘리먼트 를 클릭할때
- onmouseover : 사용자가 HTML요소위로 마우스를 이동합니다
- onmouseout : 사용자가 HTML요소에서 마우스를 멀리 이동합니다
- onkeydown : 사용자가 키보드에 키를 누릅니다
- onload : 브라우저가 페이지 로딩을 완료합니다

HTML이벤트는 브라우저가 수행하는 일이거나 사용자가 수행하는 일일수 있습니다
- 페이지 로딩이 완료되었습니다
- 입력필드가 변경되었습니다
- 버튼이 클릭되었습니다
